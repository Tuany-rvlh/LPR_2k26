# 📚 Repositório de Estudos - Desenvolvimento de Sistemas - 2k26

Este repositório foi criado como parte das atividades da disciplina de Linguagem de Programação, do curso de Desenvolvimento de Sistemas (DS), ministrada pelo professor José Andery.
Este projeto representa meu primeiro contato com Git e GitHub, sendo utilizado como ferramenta de aprendizado, organização e evolução dos meus estudos em programação.

## 🚀 Sobre o repositório

Aqui estão reunidos exercícios propostos ao longo das aulas semanais, com foco no desenvolvimento de lógica de programação e prática das linguagens estudadas:

- Aula 01: Conceitos básicos da computação - HELLO, WORLD! em C# e C++;
- Aula 02: Introdução e lógica de programação - atribuições, operadores e comandos básicos;
- Aula 03: Git e GitHub;
- Aula 04: Estruturas de seleção;
- Aula 05: Estruturas de repetição.

## 💻 Linguagens utilizadas

- C#
- C++

## 📖 Objetivos do repositório

- Registrar minha evolução na disciplina;
- Praticar o uso do Git e GitHub;
- Desenvolver lógica de programação;
- Resolver exercícios propostos em sala de aula;
- Aprimorar habilidades em C# e C++.
